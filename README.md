# Budget Ease – Personalized Budget Management Mobile Application for University Students
Budget Ease is a mobile application designed to help users track daily expenses, manage budgets, and visualize spending patterns using interactive charts.

This app also integrates machine learning models to provide intelligent budget recommendations based on user behavior.

# Features
1. User Account Management 
2. Income Management 
3. Expense Tracking  
4. Financial History
5. Analytics and Visualization 
6. Financial Summary Sheet
7. Notifications
8. Personalized Recommendations and behavior tips

# Tech Stack
# Mobile App : Flutter (Dart)

# Backend & Machine Learning:
* Python
* Flask 
* Scikit-learn
* TensorFlow / Keras

# Machine Learning Capabilities
* Predicts user spending behavior
* Recommends personalized budget plans
* Uses trained models for financial insights

#  How to Run

# Mobile App

1. Clone the repository
2. Open in VS Code or Android Studio
3. Run:
flutter pub get
flutter run

# Backend

1. Navigate to backend folder
2. Install dependencies:
pip install -r requirements.txt
3. Run:
python app.py
